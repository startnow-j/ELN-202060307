## 三、角色工作流核对

### 3.1 SUPER_ADMIN（超级管理员）工作流

```
┌─────────────────────────────────────────────────────────────────┐
│  SUPER_ADMIN 权限清单                                            │
├─────────────────────────────────────────────────────────────────┤
│  ✅ 用户管理                                                     │
│     - 创建/编辑/禁用所有角色用户（包括SUPER_ADMIN）               │
│     - 不能修改自己的角色                                         │
│     - 系统至少保留一个超级管理员                                 │
│                                                                  │
│  ✅ 文件管理                                                     │
│     - 查看所有项目文件                                           │
│     - 查看所有用户暂存区                                         │
│     - 打包下载任意项目/用户文件                                  │
│                                                                  │
│  ✅ 项目管理                                                     │
│     - 创建/编辑/查看所有项目                                     │
│     - 删除项目（仅SUPER_ADMIN）                                  │
│     - 恢复已归档项目（仅SUPER_ADMIN）                            │
│                                                                  │
│  ✅ 实验管理                                                     │
│     - 查看/编辑/删除所有实验                                     │
│     - 审核所有实验                                               │
│     - 解锁已锁定实验                                             │
└─────────────────────────────────────────────────────────────────┘
```

**测试清单**：

1. **用户管理**
   - [ ] 进入用户管理页面
   - [ ] 尝试创建新用户（选择不同角色）
   - [ ] 确认可以看到SUPER_ADMIN角色选项
   - [ ] 尝试编辑其他管理员账号
   - [ ] 尝试禁用普通用户账号

2. **项目管理**
   - [ ] 创建新项目
   - [ ] 尝试删除项目（确认按钮可用）

3. **审核管理**
   - [ ] 进入审核管理页面
   - [ ] 确认可以查看所有待审核实验

### 3.2 ADMIN（管理员）工作流

```
┌─────────────────────────────────────────────────────────────────┐
│  ADMIN 权限清单                                                  │
├─────────────────────────────────────────────────────────────────┤

│                                                                  │
│  ✅ 项目管理                                                     │
│     - 创建/编辑/查看所有项目                                     │
│     - 不能删除项目                                               │
│     - 可以归档项目                                               │
│                                                                  │
│  ✅ 实验管理                                                     │
│     - 查看/编辑所有实验                                          │
│     - 审核所有实验                                               │
│     - 解锁已锁定实验                                             │
└─────────────────────────────────────────────────────────────────┘
```

2. **项目管理**
   - [ ] 创建新项目
   - [ ] 尝试删除项目（确认按钮不可用或显示权限提示）

### 3.3 RESEARCHER（研究员）工作流

```
┌─────────────────────────────────────────────────────────────────┐
│  RESEARCHER 权限清单                                             │
├─────────────────────────────────────────────────────────────────┤
│  ❌ 用户管理                                                     │
│     - 无权访问用户管理页面                                       │
│                                                                  │
│  ❌ 文件管理                                                     │
│     - 无权访问文件管理页面                                       │
│                                                                  │
│  ✅ 项目管理                                                     │
│     - 查看自己参与的项目                                         │
│     - 不能创建项目（除非是项目创建者）                           │
│     - 项目角色：PROJECT_LEAD / MEMBER / VIEWER                  │
│                                                                  │
│  ✅ 实验管理                                                     │
│     - 创建自己的实验                                             │
│     - 编辑自己的实验（未锁定状态）                               │
│     - 提交审核                                                   │
│     - 查看自己参与的实验                                         │
└─────────────────────────────────────────────────────────────────┘
```

**测试清单**：

1. **权限验证**
   - [ ] 登录研究员账号
   - [ ] 确认侧边栏不显示"用户管理"菜单
   - [ ] 确认侧边栏不显示"文件管理"菜单
   - [ ] 确认侧边栏不显示"审核管理"菜单（除非是项目负责人）

2. **实验创建**
   - [ ] 点击"新建实验"按钮
   - [ ] 填写实验信息
   - [ ] 上传附件
   - [ ] 保存实验

3. **附件上传测试**
   - [ ] 在实验编辑页面上传文件
   - [ ] 确认文件名保留原始名称（带日期前缀）
   - [ ] 下载附件确认内容正确
