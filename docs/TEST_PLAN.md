## 测试清单

### 模块一：用户认证 (必测)

#### 1.1 登录功能
- [ ] 使用正确账号密码登录 → 应成功跳转到仪表盘
- [ ] 使用错误密码登录 → 应显示错误提示
- [ ] 使用不存在的邮箱登录 → 应显示错误提示

#### 1.2 登出功能
- [ ] 点击侧边栏底部登出按钮 → 应跳转到登录页
- [ ] 刷新页面 → 应保持登出状态

#### 1.3 用户注册（如有开放）
- [ ] 新用户注册 → 应成功创建账号
- [ ] 重复邮箱注册 → 应提示邮箱已存在

---

### 模块二：项目管理 (必测)

#### 2.1 创建项目
- [ ] 点击"项目管理" → 点击"新建项目"
- [ ] 填写项目名称、描述 → 保存成功
- [ ] 项目列表显示新建项目

#### 2.2 项目成员管理
- [ ] 进入项目详情 → 添加成员
- [ ] 设置成员角色（项目负责人/成员/观察者）
- [ ] 成员列表正确显示

#### 2.3 项目状态变更
- [ ] 结束项目 → 状态变为"已完成"
- [ ] 解锁项目 → 状态恢复为"进行中"
- [ ] 归档项目 → 状态变为"已归档"（仅超管）

---

### 模块三：实验记录管理 (核心必测)

#### 3.1 创建实验
- [ ] 点击"新建实验"
- [ ] 填写标题、摘要、结论
- [ ] 使用富文本编辑器格式化内容
- [ ] 关联项目
- [ ] 保存成功，状态为"草稿"

#### 3.2 编辑实验
- [ ] 打开已有实验
- [ ] 修改内容并保存
- [ ] 验证内容已更新

#### 3.3 附件管理
- [ ] 上传Word/PDF/Excel文件
- [ ] 预览附件内容
- [ ] 删除附件

---

### 模块四：审核流程 (核心必测)

#### 4.1 提交审核
- [ ] 在草稿实验中点击"提交审核"
- [ ] 选择审核人
- [ ] 实验状态变为"待审核"

#### 4.2 审核通过
- [ ] 使用审核人账号登录
- [ ] 在"我的任务"中看到待审核实验
- [ ] 点击审核 → 选择"通过"
- [ ] 实验状态变为"已锁定"

#### 4.3 需要修改
- [ ] 审核人选择"需要修改"
- [ ] 填写修改意见
- [ ] 实验状态变为"需要修改"
- [ ] 作者收到修改意见，修改后重新提交

#### 4.4 解锁申请
- [ ] 在已锁定实验中点击"申请解锁"
- [ ] 填写解锁原因
- [ ] 审核人批准/拒绝
- [ ] 批准后实验可编辑

---
