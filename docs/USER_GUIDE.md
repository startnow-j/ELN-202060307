## 实验记录管理

### 创建实验

1. 点击侧边栏 **"实验记录"**
2. 点击右上角 **"新建实验"** 按钮
3. 填写实验信息：
   - **标题**：实验名称（必填）
   - **摘要**：实验目的和方法概述
   - **结论**：实验结果和发现
   - **关联项目**：选择所属项目
   - **标签**：添加分类标签
4. 点击 **"保存"** 或 **"提交审核"**

### 编辑实验

1. 在实验列表中点击实验卡片
2. 进入实验详情页
3. 点击 **"编辑"** 按钮
4. 修改内容后保存

> **注意**：已提交审核或已锁定的实验不能直接编辑

### 上传附件

1. 在实验编辑页面，找到 **"附件"** 区域
2. 点击 **"上传"** 或拖拽文件
3. 支持的格式：
   - 文档：.doc, .docx, .pdf
   - 数据：.xls, .xlsx, .csv
   - 图片：.jpg, .png, .gif
   - 其他：.txt, .zip 等

### 使用富文本编辑器

编辑器工具栏包含：

| 功能 | 说明 |
|------|------|
| 标题 | 设置标题级别 |
| 加粗/斜体 | 文字格式 |
| 列表 | 有序/无序列表 |
| 表格 | 插入表格 |
| 代码块 | 插入代码 |
| 图片 | 插入图片 |
| 链接 | 添加超链接 |

## 审核流程

### 实验状态说明

| 状态 | 颜色 | 说明 |
|------|------|------|
| 草稿 | 灰色 | 正在编辑中，可随时修改 |
| 待审核 | 黄色 | 已提交，等待审核人审核 |
| 需要修改 | 橙色 | 审核未通过，需要修改后重新提交 |
| 已锁定 | 绿色 | 审核通过，不可修改 |

### 提交审核

1. 在实验详情页，确认内容已完善
2. 点击 **"提交审核"** 按钮
3. 选择审核人（可选）
4. 填写提交说明（可选）
5. 确认提交

### 审核实验（审核人）

1. 进入 **"我的任务"** → **"待我审核"**
2. 点击待审核的实验
3. 查看实验内容
4. 选择审核操作：
   - **通过**：实验锁定，流程结束
   - **需要修改**：填写修改意见，退回作者
   - **转交**：转给其他审核人

### 申请解锁

如果需要修改已锁定的实验：

1. 进入已锁定的实验
2. 点击 **"申请解锁"** 按钮
3. 填写解锁原因
4. 等待审核人审批
5. 审批通过后可编辑

## 项目管理

### 创建项目

1. 点击侧边栏 **"项目管理"**
2. 点击 **"新建项目"**
3. 填写项目信息：
   - 名称（必填）
   - 描述
   - 预计结束日期
4. 保存

### 项目成员管理

1. 进入项目详情页
2. 点击 **"成员"** 标签
3. 添加成员：
   - 选择用户
   - 设置角色（项目负责人/成员/观察者）
4. 保存

### 项目状态

| 状态 | 说明 |
|------|------|
| 进行中 | 项目正在进行 |
| 已完成 | 项目已结束，实验锁定 |
| 已归档 | 项目归档，仅超级管理员可恢复 |

## 视角模式

### 什么是视角模式？

视角模式决定了您在系统中能看到的数据范围。系统提供两种视角模式：

| 视角模式 | 数据范围 | 适用角色 |
|---------|---------|---------|
| **默认视角** | 仅显示您参与的数据 | 所有用户 |
| **全局视角** | 显示系统中所有数据 | 管理员、超级管理员 |

### 如何切换视角？

管理员和超级管理员可以在页面右上角找到视角切换开关：

1. 找到右上角的 **"全局视角"** 开关
2. 打开开关 → 切换到全局视角
3. 关闭开关 → 切换回默认视角

> 研究员角色看不到此开关，始终使用默认视角

### 各模块视角差异

#### 实验记录

| 视角 | 可见实验 |
|------|---------|
| 默认视角 | 您创建的实验 + 您参与项目的实验 |
| 全局视角 | 系统中所有实验 |

**使用场景**：
- **默认视角**：日常工作时，专注于自己的实验
- **全局视角**：需要审核、监督或查看他人实验时

#### 项目管理

| 视角 | 可见项目 |
|------|---------|
| 默认视角 | 您创建的项目 + 您参与的项目 |
| 全局视角 | 系统中所有项目 |

**使用场景**：
- **默认视角**：管理自己的项目
- **全局视角**：监督所有项目进展，跨项目管理

#### 我的任务

| 任务类型 | 默认视角 | 全局视角 |
|---------|---------|---------|
| 我的草稿 | 自己的草稿 | 不变 |
| 待我审核 | 指定审核人 + 负责项目的待审核 | 所有待审核实验 |
| 需要修改 | 自己被退回的实验 | 不变 |
| 解锁申请 | 自己的申请 | 待处理的他人申请 |

**使用场景**：
- **默认视角**：处理自己的工作
- **全局视角**：主动处理系统中的待审核任务

#### 审核流程

| 功能 | 默认视角 | 全局视角 |
|------|---------|---------|
| 提交审核 | 只能提交自己的实验 | 可代理提交任何实验 |
| 审核操作 | 审核指定/项目的实验 | 可审核任何实验 |
| 解锁审批 | 项目相关的解锁申请 | 可审批任何解锁申请 |

### 视角使用建议

#### 何时使用默认视角？

- 日常实验记录工作
- 管理自己的项目
- 专注于个人任务时
- 避免被无关信息干扰

#### 何时切换到全局视角？

- 需要审核他人实验时
- 查看实验室整体工作进展
- 跨项目管理或监督
- 处理系统级任务（用户管理、AI配置等）

### 注意事项

1. **视角只影响数据可见范围**，不改变操作权限
2. **全局视角下仍需遵守权限规则**（如不能编辑他人已锁定的实验）
3. **切换视角后会刷新数据**，请确保已保存当前工作
4. **超级管理员**始终拥有最高权限，但也可使用默认视角专注工作

## 角色权限说明

### 研究员 (RESEARCHER)

- ✅ 创建和编辑自己的实验
- ✅ 查看所属项目的实验
- ✅ 提交审核/申请解锁
- ❌ 审核他人实验
- ❌ 用户管理

### 管理员 (ADMIN)

- ✅ 研究员所有权限
- ✅ 全局视角查看所有实验
- ✅ 审核实验
- ✅ 解锁实验
- ✅ 用户管理（不能管理超级管理员）

### 超级管理员 (SUPER_ADMIN)

- ✅ 管理员所有权限
- ✅ 管理所有用户（包括超级管理员）
- ✅ 删除项目
- ✅ 恢复已归档项目
- ✅ AI配置管理

## 常见问题

### Q: 为什么我看不到某些实验？

A: 可能原因：
1. 实验不在您参与的项目中
2. 如果您是管理员，请切换到"全局视角"

### Q: 实验被锁定了怎么办？

A: 如果需要修改已锁定的实验，请使用"申请解锁"功能，填写原因后等待审核人审批。

### Q: 如何切换全局视角？

A: 管理员和超级管理员可以在页面右上角找到视角切换开关：
- **默认视角**：仅显示您参与的实验
- **全局视角**：显示所有实验

详细说明请参考上方"视角模式"章节。

### Q: 为什么我在全局视角下仍然无法编辑某个实验？

A: 视角只影响数据可见范围，不改变操作权限。可能的原因：
1. 实验已被锁定（审核通过），需要申请解锁
2. 实验状态不允许编辑
3. 您没有编辑他人实验的权限

### Q: 忘记密码怎么办？

A: 请联系系统管理员重置密码。
